new in 0.5:
  • integration with Outlook: Get the VBA Macro, open the Visual Basic Editor (menu Tools->Macro), import the macro file and close the editor. Then customize your Outlook toolbar, add a button calling the PlataneSync Macro. Click on it regularly!
  • project view
  • integrated lpsolve, very fast native solver
  • fixed charset problems

new in 0.4:
  • super tasks, example: the load of task PROJECT-nnn will be limited by the load of task PROJECT
  • group tasks, example: tasks PROJECT [1], PROJECT [2] will be automatically disjoint in time and collapsed in view
  • added absence flag and special scheduling rules (no overflow, remaining used as base for other tasks)
  • fixed many small scheduling bugs

new in 0.3:
  • now tasks can be specified with an effort and a load
  • integration with jira (currently read-only)
  • fixed bug limiting load-based task to 90 days
  • known bug: load-based tasks with from/to are not accurately scheduled
